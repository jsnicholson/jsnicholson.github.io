Task Brief

This coursework was focused on improving our C++ skill by building on top of a pre-existing framework

Requirements included points such as

  • A polymorphic state system to swap between different 'scenes'
  • Having a functional pause screen
  • Saving and loading data
  • Dynamic object creation and destruction
  • Capture key and mouse inputs
  • Animated objects
  • An all round polished game

Features

Polymorphic state system

  • Menu
  • Game
  • Highscore
  • Settings

Animated objects

  • Sprite animations for plane and missiles
  • Bouncing text and image

Smooth transitions between scenes (eg. menu to game)

Complex missile spawning

  • Game gets more difficult over time
  • More frequent spawns, faster speeds
  • Every 3rd missile comes directly at the player to keep them moving

Console for commands

  • drawboxes to draw bounding boxes of all objects
  • god to disable collision detection

Score saving

  • Top 10 scores saved
  • New highscore image shown

Settings screen

  • Changeable key bindings
