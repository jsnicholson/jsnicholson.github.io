Aim

Browsing forum websites such as Reddit, I often run into albums hosted on Imgur

The inbuilt album viewer is usually a simple scroll through one entry at a time

However, particularly for 'inspiration' or collage albums, this interface didnt suit, hence I built this gallery-type viewer

My only real requirement was to have a responsive grid view for imgur albums but as time has passed, the feature list has grown

Features

Gallery viewing of images from Imgur

Load images from an album or from an account

View in different orders

  • Oldest - Newest
  • Newest - Oldest
  • Random

View varying amounts of images

Display as a fullscreen slideshow

Analysis

This is a project that only very barely meets requirements. The code is most certainly subpar but it hits the primary feature I wanted of it In that sense it is a success but just barely.

I will most likely continue to work on this project as I see fit, adding more features should they be needed.

Regardless of its success as an application, it most definitely served as a great introduction to web API and OAuth2 (despite this being a poor implemtation of OAuth partly thanks to my hosting choices)
