Current Features

Super Rotation System (SRS)

  • Take a look at the wiki for more information
  • Currently have both wall kicks and floor kicks implemented

Colour palette swapping

  • 4 grayscale shades are used and then swapped at runtime using a shader
  • Variety of palettes currently created with easy addition

Tetromino Builder to create tetromino's based on any given tile

  • This will allow for unlockable tiles for further customisation later

Planned Features

Holding keys for continued movement

Row clearing on landing

Scoring

  • Points for rows cleared

Game over screen
